# 設計文件

## 山手線列車模擬器 - 詳細設計規格

### 1. 列車系統設計

#### 物理模型
- **最高速度**：80 km/h
- **加速度**：1.5 m/s²
- **減速度**：2 m/s²
- **停靠時間**：15 秒

#### 狀態機制
```
[加速] → [正常行駛] → [減速] → [停靠] → [開門] → [關門] → [加速]
```

### 2. 車站系統設計

#### 山手線結構
- **總車站數**：29 個
- **環線時間**：約 60 分鐘
- **車站間距**：平均 2-3 公里
- **主要轉乘站**：新宿、澀谷、池袋、上野、東京

#### 車站類型
- **Terminal**：終站（東京站）
- **Major**：主要車站（新宿、澀谷等）
- **Standard**：普通車站

### 3. 乘客系統設計

#### 時間段設置
- **高峰時段**：07:00-09:00, 17:00-19:00
  - 乘客容量：150-200 人
  - 擁擠度：70-100%
  
- **離峰時段**：14:00-16:00
  - 乘客容量：30-100 人
  - 擁擠度：15-50%
  
- **夜間時段**：22:00-06:00
  - 乘客容量：0-30 人
  - 擁擠度：0-15%

#### 乘客行為
- 站立、坐著、使用手機、睡覺
- 上下車比例根據時間自動調整

### 4. UI/UX 設計

#### 主要顯示信息
1. **列車狀態**
   - 當前速度（km/h）
   - 現在位置（車站名）
   - 下一車站

2. **乘客信息**
   - 目前乘客數 / 最大容量
   - 擁擠度等級（空車 → 正常 → 滿員）

3. **時間信息**
   - 遊戲時間（HH:MM）
   - 日夜循環指示

4. **路線圖**
   - 動態山手線地圖
   - 當前位置標記
   - 下一車站提示

#### 互動元素
- 遊戲模式選擇按鈕
- 座位選擇界面
- 車門手動控制
- 語言切換（日語/英語）

### 5. 音頻系統設計

#### 聲音分類
1. **列車音效**
   - 加速音（漸進式）
   - 制動音（漸進式）
   - 門開音
   - 門關音
   - 車輪聲

2. **播報系統**
   - 車站播報（日語 + 英語）
   - 下一車站提示
   - 警告聲音

3. **環境音**
   - 車站環境聲
   - 乘客聲（談話、雜音）
   - 風聲
   - 外部交通聲

4. **背景音樂**
   - 日間音樂
   - 夜間音樂
   - 菜單音樂

### 6. 視覺設計

#### 美術風格
- 日本寫實風格
- 低多邊形 + 高質量紋理
- 準確的建築細節

#### 色彩方案
- **白天**：明亮、清爽
- **黃昏**：溫暖的橙紅色
- **夜間**：深藍、霓虹燈

### 7. 遊戲模式

#### 模式 1：遊客模式
- **目的**：教育和探索
- **功能**：
  - 車站信息和景點介紹
  - 歷史故事
  - 建築細節說明

#### 模式 2：通勤者模式
- **目的**：逼真的日常體驗
- **功能**：
  - 確實的時間表
  - 尖峰時段模擬
  - 日常習慣

#### 模式 3：自由探索
- **目的**：自由體驗
- **功能**：
  - 可在任何車站上下車
  - 360度環境探索
  - 拍照功能

#### 模式 4：挑戰模式
- **目的**：遊戲化體驗
- **功能**：
  - 完成特定任務
  - 收集徽章
  - 排行榜
